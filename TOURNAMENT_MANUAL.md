## 1. Что такое HardPoint Series

HardPoint Series — еженедельный турнир по CS2 для посетителей клуба HardPoint Szczecin. Турнир проводится **в клубе на LAN**, матчи играются через платформу **FACEIT** (античит, серверы, статистика).

**Форматы:**
- **5×5** — полная команда, 5 игроков
- **2×2** — дуо, 2 игрока

**Призы:**
- 🥇 1 место — 8 часов игры (промокод)
- 🥈 2 место — 5 часов игры (промокод)
- 🥉 3 место — 3 часа игры (промокод)

Призы выдаются в виде промокодов, которые можно использовать при первом визите в клуб.

## 4. Расписание турнира

```
ПОНЕДЕЛЬНИК 04:00  →  Бот автоматически генерирует сетку
                       Капитаны получают уведомления
                       Live сетка открывается всем
```

> ⚠️ Всё происходит **автоматически**. Администратор не должен ничего запускать вручную.

## 5. Регистрация команд

### Шаг 1 — Игрок указывает FACEIT nickname

Каждый участник (включая капитана) обязан указать свой FACEIT nickname в боте:

1. Открыть Mini App → раздел **Series**
2. Вкладка **Заявка**
3. Ввести FACEIT nickname → нажать **Сохранить**
4. Бот проверяет что такой ник существует на FACEIT
5. Без подтверждённого FACEIT ника — нельзя создать или вступить в команду

### Шаг 2 — Капитан создаёт команду

1. Раздел Series → Заявка → **Создать команду**
2. Выбрать формат: 5×5 или 2×2
3. Ввести название команды → **Подать заявку**
4. Команда попадает на проверку администратору

### Шаг 3 — Администратор принимает команду

1. Открыть Mini App → Series → вкладка **Админ**
2. Список заявок — проверить название и формат
3. Нажать **Принять** или **Отклонить**

> После принятия команда появляется в таблице и может регистрироваться на турнир.

### Шаг 4 — Игроки вступают в команду

Капитан делится кодом команды (`HL-XXXXXX`):

1. Series → Заявка → кнопка **Пригласить** рядом с командой
2. Скопировать или отправить код в Telegram
3. Игрок открывает Mini App → Series → Заявка → вводит код → **Вступить**

> ⚠️ Игрок должен сначала указать FACEIT nickname, иначе вступить не получится.

### Сколько команд нужно для турнира

| Зарегистрировалось | Сетка | Резерв |
|---|---|---|
| 4 команды | 4 команды | — |
| 5–7 команд | 4 команды | остальные в резерве |
| 8 команд | 8 команд | — |
| 9–15 команд | 8 команд | остальные в резерве |
| 16 команд | 16 команд | — |

Команды попадают в сетку по **приоритету регистрации** — кто первым зарегистрировался, тот играет.

Если зарегистрировалось **меньше 4 команд** — турнир не стартует, бот отправляет уведомление в группу, регистрация продлевается.

## 6. Генерация сетки

**Происходит автоматически** в понедельник в 04:00 по варшавскому времени.

Что делает бот:
1. Берёт первые 4/8/16 команд по приоритету
2. Случайная жеребьёвка — перемешивает команды
3. Формирует пары Round 1
4. Создаёт пустые слоты для следующих раундов
5. Постит уведомление в группу игроков
6. Отправляет каждому капитану личное уведомление с составом пары
7. Создаёт тред в группе капитанов для каждой пары

**Live сетка** доступна всем сразу после генерации:
- Mini App → кнопка ⚡ **LIVE** в нижнем меню
- Показывает все раунды, счёт, статусы матчей
- Обновляется каждые 30 секунд автоматически

## 7. FACEIT — создание комнаты матча

После того как капитаны договорились о времени матча, нужно создать комнату в FACEIT Hub. Есть два способа:

---

### Способ A — Капитан создаёт через бота *(рекомендуется)*

1. Капитан открывает Mini App → LIVE сетка
2. Находит свой матч → нажимает **"Создать комнату FACEIT"**
3. Бот автоматически создаёт матч в FACEIT Hub через API
4. Оба капитана получают ссылку на комнату в личные сообщения
5. Игроки заходят по ссылке и начинают матч

---

### Способ B — Администратор создаёт вручную в FACEIT

1. Зайти на https://organizer.faceit.com
2. Выбрать HardPoint Hub → **Matches** → **Create match**
3. Выбрать формат (5v5 или 2v2)
4. Добавить игроков обеих команд (по FACEIT никам)
5. Сохранить матч

**После создания вручную:**
- Бот проверяет Hub каждые 5 минут
- Находит новый матч → сравнивает игроков с базой данных
- Если совпадают FACEIT ники игроков команды → прикрепляет автоматически
- Уведомляет капитанов ссылкой на комнату

**Три сценария сопоставления:**

| Ситуация | Действие бота |
|---|---|
| Все игроки команды совпали | Прикрепляет автоматически, уведомляет капитанов |
| Частичное совпадение | Отправляет запрос администратору на подтверждение |
| Совпадений нет | Администратор вставляет ссылку вручную |

## 8. Проведение матчей

### В день матча

1. Команды приходят в клуб в договорённое время
2. Открывают ссылку на FACEIT комнату (из уведомления бота)
3. Все игроки заходят в комнату на FACEIT
4. Матч начинается автоматически когда все готовы
5. Играют на картах FACEIT (античит активен)

### Формат

- **5×5** — один матч, BO1 (лучший из 1)
- **2×2** — один матч, BO1

### Дедлайн

Все матчи должны быть сыграны до **воскресенья 23:59** по варшавскому времени.

Если команда не вышла на матч — соперник получает техническую победу. Администратор вносит результат вручную.

## 9. Результаты и сетка

### Автоматически (через FACEIT API)

После завершения матча на FACEIT:
1. Бот получает результат автоматически
2. Обновляет счёт в Live сетке
3. Победитель переходит в следующий раунд автоматически
4. Обе команды получают уведомление

### Вручную (запасной вариант)

Если автоматика не сработала:
1. Капитан открывает Mini App → LIVE → находит свой матч
2. Нажимает **"Внести результат"**
3. Вводит счёт (например 16:10)
4. Бот обновляет сетку

> Результат может внести только капитан одной из команд или администратор.

## 10. Нестандартные ситуации

### Команда не пришла на матч
1. Зафиксировать факт неявки в треде группы капитанов
2. Администратор открывает Mini App → LIVE → матч → **"Внести результат"**
3. Вводит счёт 1:0 в пользу присутствующей команды

### Технические проблемы с FACEIT
1. Скриншот ошибки в треде капитанов
2. Администратор создаёт новую комнату вручную
3. При необходимости — перенести матч

### Спорный результат
1. Обе команды присылают скриншоты результата в тред
2. Администратор принимает решение
3. Вносит финальный счёт вручную через Mini App

### Бот не сгенерировал сетку
Проверить на сервере:
```bash
ssh -i ~/Documents/telegram/.oracle-ssh-key ubuntu@138.2.133.215
sudo journalctl -u hardpoint-bot -n 50 --no-pager
```
Если бот упал — перезапустить:
```bash
sudo systemctl restart hardpoint-bot
```

### Команда хочет перенести матч
- Каждая команда имеет право на **1 перенос** за турнир
- Оба капитана должны согласиться в треде группы капитанов
- Матч должен быть сыгран до воскресенья 23:59
